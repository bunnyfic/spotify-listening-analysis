# Spotify Listening Analysis 

This project analyzes my personal Spotify streaming history to understand
listening patterns, artist preferences, and time-based behavior.

## Dataset
- Spotify streaming history (past 1 year)
- Data provided by Spotify under Account Data

## Analysis Performed
- Top artists and most played tracks
- Listening behavior by hour and day
- Monthly listening trends

## Machine Learning Extension
- Logistic Regression to classify listening time as Day or Night
- Features used: listening duration and hour of day

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Note
Raw Spotify data is not shared to protect privacy.
